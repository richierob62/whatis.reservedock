fool-proof rules to maintain control

easily set rules that affect which docks are available

By simply setting values for a few parameters, you can create a constraint (a set of rules) which can be applied to any dock. Your constraint can be based on any combination of:

  • The day of the week
  • The time of day
  • The effective dates
  • The time to unload
  • The shipment contents
  • On pallets or not
  • The packaging
  • The shipper/vendor
  • The type of vehicle
